# CdCore's Blog

CandyCoreBlog 是我的个人博客，记录前端开发、工程实践与随想杂记。

## Tech Stack

- Astro + React
- Tailwind CSS
- TypeScript

## Installation

```bash
pnpm install
```

## Local Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

产物输出到 `dist` 目录。

## Preview

```bash
pnpm preview
```

## Deployment

使用 GitHub Actions 自动部署到 GitHub Pages：推送到 `main` 分支即触发构建与发布。
