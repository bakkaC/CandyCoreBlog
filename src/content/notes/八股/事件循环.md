浏览器：

```
1. 从宏任务队列取出一个任务执行
2. 执行过程中产生的微任务放入微任务队列
3. 当前宏任务执行完后
4. 立即清空整个微任务队列,微任务产生微任务也会加入队尾清空
5. （可能进行渲染）
6. 再执行下一个宏任务
```

宏任务（Task）

- script（整体代码）
- setTimeout
- setInterval
- UI 事件
- postMessage

微任务（Microtask）

- Promise.then / catch / finally
- MutationObserver
- queueMicrotask

Node：

```
1. timers            (setTimeout / setInterval)
2. pending callbacks
3. idle, prepare     (内部用)
4. poll              (I/O)
5. check             (setImmediate)
6. close callbacks
```

Node 有两种“微任务”：

1. process.nextTick（优先级最高）
2. Promise.then（普通微任务）

优先级：`process.nextTick > Promise.then`
每个阶段结束后都会：
-  **先清空 nextTick 队列**  
- **再清空 Promise 微任务队列**
